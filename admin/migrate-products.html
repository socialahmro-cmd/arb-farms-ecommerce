Migrate products-db.js → Firestore

One-time (idempotent) script: writes every product in window.productsDb into the products collection, using each product's id as the document ID. The tags field is intentionally left out for now — everything else (sku, name, weight, price, category, image, hoverImage, etc.) is written as-is. Safe to re-run — it overwrites existing docs with the current file contents rather than duplicating them.
